# Blind Person Reader and Language Translator System

## Overview

The Blind Person Reader and Language Translator System is an AI-powered assistive solution developed to help visually impaired people read printed text and understand it in their preferred language. The system captures text using a camera, extracts it with Optical Character Recognition (OCR), translates it into multiple languages, and converts the translated text into speech.

This project combines Artificial Intelligence, Computer Vision, Natural Language Processing, and Embedded Systems to provide an affordable and portable smart reading assistant.

---

## Features

- Capture text using a camera
- Optical Character Recognition (OCR)
- Multi-language translation
- Text-to-Speech (TTS)
- Voice output through speaker/headphones
- Supports visually impaired users
- Portable Raspberry Pi based system
- Real-time text reading
- Easy-to-use interface

## Project Workflow

1. Capture image using Raspberry Pi Camera.
2. Detect printed text using OCR.
3. Extract text from the image.
4. Translate text into the selected language.
5. Convert translated text into speech.
6. Play audio through the speaker.
